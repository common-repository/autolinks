Autolinks is a very smart wordpress plugin, it automatically generate a hyper links for images of the post summary.
For category list,search result etc, if topic summary have images, autolinks will generate hyper links for these images, 
these hyper links will link to the topicâ€™s full text.
Additionally, Autolinks can also automatically generate title and alt information for images when they are lack of them.
Thus, Autolinks can improve both human friendly and search engine friendly.

== Installation ==

1. Unpack the *.zip file and extract the /autolinks/ folder.Drop the 'autolinks' folder into your 'wp-content/plugins' folder.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setup options through 'Settings/AutoLinks' menu in WordPress.
